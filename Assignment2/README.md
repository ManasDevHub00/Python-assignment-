## Task 1: Check Whether a Number is Even or Odd

### Functionality

This program determines whether a number entered by the user is even or odd.
The user inputs an integer, and the program checks divisibility by 2 using the modulus (`%`) operator. If the remainder is zero, the number is even; otherwise, it is odd. The result is displayed to the user.

### Explanation

* Accepts input from the user.
* Converts input into an integer.
* Uses an `if-else` condition to check parity.
* Prints whether the number is even or odd.

## Task 2: Sum of Integers from 1 to 50 Using a Loop

### Functionality

This program calculates the sum of all integers from 1 to 50 using a `for` loop.
A variable is initialized to store the total sum, and each number in the given range is added sequentially. The final calculated sum is displayed as output.

### Explanation

* Initializes a variable to store total sum.
* Uses `range(1, 51)` to generate numbers from 1 to 50.
* Iterates using a `for` loop.
* Adds each number to the total.
* Prints the final result.
