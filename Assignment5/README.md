# Python Basic Programs

This repository contains simple Python programs demonstrating the use of **dictionaries, lists, and list operations**.

## Task 1: Student Marks Dictionary

### Description

This program creates a dictionary where **student names are keys** and **their marks are values**.
The user enters a student’s name, and the program retrieves and displays the corresponding marks.
If the name does not exist in the dictionary, an appropriate message is shown.

### Features

* Uses Python **dictionary**
* Takes **user input**
* Checks if a key exists in the dictionary

### Example Output

Enter the student's name: Manas
Marks of Manas : 88

If the student is not found:
Student not found in the record.

## Task 2: List Extraction and Reversal

### Description

This program demonstrates **list creation, slicing, and reversing**.

Steps performed:

1. Creates a list of numbers from **1 to 10**.
2. Extracts the **first five elements** using list slicing.
3. Reverses the extracted elements.
4. Prints both the extracted list and the reversed list.

### Example Output

Extracted list: [1, 2, 3, 4, 5]
Reversed list: [5, 4, 3, 2, 1]
