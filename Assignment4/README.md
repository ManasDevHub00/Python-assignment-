# Python File Handling Assignment

## Overview

This assignment demonstrates basic **file handling in Python**.
It includes reading files, writing data to files, appending data, and handling errors.

---

## Task 1: Read a File and Handle Errors

In this task, the program:

1. Opens a text file named **Sample.txt** from the folder **Assignment4**.
2. Reads the file **line by line**.
3. Prints each line on the screen.
4. Handles the error if the file **does not exist** using `try` and `except`.

This prevents the program from crashing and displays an error message.

---

## Task 2: Write, Append, and Read a File

In this task, the program performs three operations:

1. **Write Data**

   * Takes input from the user.
   * Writes the input to a file named **output.txt**.

2. **Append Data**

   * Takes additional input from the user.
   * Appends the new data to the same file without deleting existing data.

3. **Read File**

   * Opens the file.
   * Reads the final content line by line.
   * Displays the content on the screen.

---

## Concepts Used

* File Handling in Python
* `open()` function
* File modes (`r`, `w`, `a`)
* `with` statement
* `try` and `except` error handling
* `FileNotFoundError`

---

## Conclusion

This assignment helps understand how to **read, write, append, and handle errors in files using Python**, which is an important concept in programming.
